# FIT-BIT-IPK-2020-proj1

První projekt do předmětu Počítačové komunikace a sítě na FIT VUT v Brně 2020.

## Použité technologie

 - C++11
 - POSIX sockety

## Sestavení aplikace

```bash
make build
```

## Spuštění aplikace

```bash
make run PORT=5353
```

Spustí aplikaci na portu 5353.
